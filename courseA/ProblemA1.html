課題1-(1)の概要

この Clockプログラムは，実際のアナログ時計と一部で動作が異なっています．
通常のアナログ時計では，時間の経過にしたがって短針は少しづつ動いていきますが，この Clockプログラムでは短針が1時間に1回しか動きません．
そこで，Clockプログラムの動作をできる限り実際のアナログ時計に近付けるために，短針も1分に1回動くよう針の更新機能を変更することを考えます．

ただし現在の設計では，以下のようにMin オブジェクトが Min_ang オブジェクトを，Hour オブジェクトが Hour_ang オブジェクトを
それぞれ独立に更新するようになっており，このような機能変更に柔軟に対応することができません．
新しい機能では，Hour_ang が保持している短針の角度を計算するために，Hour オブジェクトと Min オブジェクトの両方の情報が必要になるからです． 

public class Hour {
	private Hour_ang hour_ang;
	private double value;
	  :
	public void updateMin(double min) {
		  :
		this.hour_ang.updateHour(this.value);
	}
}

public class Hour_ang {
	private double value;
	  :
	public void updateHour(double hour) {
		this.value = ((hour/6)*Math.PI);
	}
}

そこで，図のように新たに Presenter オブジェクトを導入して， このオブジェクトが Min オブジェクトや Hour オブジェクトから，
Min_ang オブジェクトや Hour_ang オブジェクトへの更新を一括して行えるように設計変更を行います．
