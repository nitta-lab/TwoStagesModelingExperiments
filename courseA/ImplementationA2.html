【Audio】
機能追加課題

課題の概要：

本課題では，

「先ほどツール上で作成したモデルから出力されたプロトタイプを用いて，リファクタリング課題同様の追加機能実装を行うこと」

を目的とし，以下の詳細な手順にしたがい課題に取り組んでいただきます．

課題概要：

この Audio プログラムに，オーディオのミュート機能を付けることを考えます．

現在の設計では，AudioVolume オブジェクトからオーディオ全体の音量，SoundVolume オブジェクトからサウンドの音量，
BGMVolume オブジェクトからBGMの音量を取得し，OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトの内部で
これらの値を使って出力の音量を計算しています．

ここで，オーディオにミュート機能を持たせるとすれば AudioVolume オブジェクト辺りが適当と考えられますが， AudioVolume オブジェクトに機能を追加した場合， OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトの両方の実装を書き換えなければなりません．

OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトのそれぞれの内部で，以下のように出力の音量の計算を別々に行っているためです．
        public class OutputSoundVolume {
            private SoundVolume soundVolume;
            private AudioVolume audioVolume;
              :
            public double getValue() {
                return (this.audioVolume.getValue()*this.soundVolume.getValue());
            }
        }
        
        public class OutputBGMVolume {
            private BGM BGMVolume;
            private Audio audioVolume;
              :
            public double getValue() {
                return (this.audioVolume.getValue()*this.BGMVolume.getValue());
            }
        }

そこで，OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトの内部で行っている出力の音量の計算を
AudioVolume オブジェクトの中で一括して行うように設計変更を行います．
