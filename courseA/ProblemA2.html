課題2-(1)の概要

この Audio プログラムに，オーディオのミュート機能を付けることを考えます．

現在の設計では，以下のように OutputSoundVolume オブジェクトと OutputBGMVolume オブジェクトが， それぞれの getValue() メソッド内部で，AudioVolume オブジェクトからオーディオ全体の音量，SoundVolume オブジェクトからサウンドの音量， BGMVolume オブジェクトからBGMの音量を取得し，取得した値を使ってそれぞれの最終出力の音量を計算しています．

public class OutputSoundVolume {
	private SoundVolume soundVolume;
	private AudioVolume audioVolume;
	  :
	public double getValue() {
		return (this.audioVolume.getValue()*this.soundVolume.getValue());
	}
}
public class OutputBGMVolume {
	private BGMVolume BGMVolume;
	private AudioVolume audioVolume;
	  :
	public double getValue() {
		return (this.audioVolume.getValue()*this.BGMVolume.getValue());
	}
}

ここで，このオーディオにミュート機能を追加することを考えた場合，AudioVolume オブジェクト辺りに追加の実装をするのが適当と考えられますが， たとえば AudioVolume オブジェクトに isMute() メソッドを追加したとすると， それに伴って OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトの両方の getValue() メソッドの実装を，たとえば以下のように書き換えなければならなくなります． 

public class OutputSoundVolume {
	private SoundVolume soundVolume;
	private AudioVolume audioVolume;
	  :
	public double getValue() {
		if (this.audioVolume.isMute()) return 0.0;
		return (this.audioVolume.getValue()*this.soundVolume.getValue());
	}
}
public class OutputBGMVolume {
	private BGMVolume BGMVolume;
	private AudioVolume audioVolume;
	  :
	public double getValue() {
		if (this.audioVolume.isMute()) return 0.0;
		return (this.audioVolume.getValue()*this.BGMVolume.getValue());
	}
}

そこで，OutputSoundVolume オブジェクトと OutputSoundVolume オブジェクトの内部で個別に行っている出力の音量の計算を AudioVolume オブジェクトの中で一括して行うように設計変更を行って，ミュート機能の実装が複数個所に散らばらないようにします．
