GitHub Repository

こちら です．

クローンしてお手元の開発環境でビルドし，動作環境を行ってください．

【Audio】 は Gradle プロジェクトとして作成しています．

    Eclipse 上でビルドする場合は，プロジェクトを右クリック→「構成」→「Gradle ネーチャーの追加」でGradleを動かせるようにしておいてください．
プログラムを実行する際は，Gradle タスクの中から「application/run」を実行してください．

【Audio】 のテストコードは /src/test/java/MainTest.java にあります．JUnitでテストすることができます．
